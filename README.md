> Web application for maintaining a list of user tasks in the form of a game
> Developed on PHP using the Laravel framework.

## Install

1. Clone repository\
   `git clone https://github.com/Mutad/Todo`
2. Install dependencies\
   `composer install`
3. Copy and setup .env\
   `cp .env.example .env`
4. Migrate db\
   `php artisan migrate`
5. test it out\
   `php artisan serve`
