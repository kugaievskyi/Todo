<h1 align="center">Todo</h1>
<p>Todo application with reward system</p>
<h2>How to setup</h2>


1. Clone repository\
   `git clone https://github.com/Mutad/Todo`
2. Install dependencies\
   `composer install`
3. Copy and setup .env\
   `cp .env.example .env`
4. test it out\
   `php artisan serve`
